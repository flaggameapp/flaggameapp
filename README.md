# 🌍 Flag Game

Learn the flags of the world through a fun, modern and progressively challenging experience.

Flag Game is an open source geography game focused on helping players recognize and memorize the flags of the world's 195 countries. Whether you're just getting started or already know your way around the globe, the game offers multiple ways to learn, practice and improve over time.

## ✨ Features

- 🌎 Play by continent
- 🌍 Challenge yourself with all 195 countries
- ⌨️ Expert Mode: type the country name instead of choosing from options
- 🏆 Achievement system
- 📊 Local statistics and progress tracking
- 🎯 Share challenge codes with friends
- 🖼️ Generate and share result images
- 🌐 Available in 20 languages
- 💾 No account required
- ⚡ Designed to work locally with offline-friendly core features

## 🎮 Game Modes

### By Continent

Learn one region at a time.

### World Mode

Choose between 10, 20, 50 or all 195 flags.

### Expert Mode

No multiple-choice answers. Type the correct country name and put your knowledge to the test.

## 📈 Your Progress

Flag Game keeps your progress on your device, including:

- Games played
- Accuracy
- Best streak
- Favorite game mode
- Completed continents
- Achievements

No registration is required.
