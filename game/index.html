Política de Privacidade

Esta política descreve como o Flag Game trata informações no aplicativo.

Nenhum dado pessoal é coletado O Flag Game não solicita, armazena ou envia dados pessoais.
Não existe login O aplicativo não possui conta de usuário, cadastro ou autenticação.
Não existe rastreamento Não usamos cookies, analytics, identificadores de publicidade ou rastreadores.
Recordes apenas locais Os recordes ficam salvos somente no dispositivo, usando armazenamento local.
Funciona offline As principais funcionalidades do aplicativo continuam disponíveis sem internet.
