## FAQ

Pergunta:

O Flag Game tem as bandeiras de todos os paises?

Resposta:

O jogo trabalha com uma base de 195 paises confirmada no codigo atual.

Pergunta:

Preciso criar conta para jogar?

Resposta:

Nao. A versao atual funciona sem cadastro, login ou conta de usuario.

Pergunta:

O progresso fica salvo?

Resposta:

Sim. O perfil local salva partidas jogadas, acertos, media, melhor sequencia, tempo medio, modo favorito e continentes concluidos neste dispositivo.

Pergunta:

Funciona offline?

Resposta:

As funcoes principais foram projetadas para funcionar de forma local/offline quando os arquivos do jogo estao disponiveis no dispositivo ou ambiente em que o jogo foi aberto.

Pergunta:

Quais modos existem?

Resposta:

Voce pode jogar por continente, no mundo inteiro com 10, 20, 50 ou 195 bandeiras, ou no modo experiente digitando o nome dos paises.

Pergunta:

Existe ranking online?

Resposta:

Nao como recurso publico ativo nesta versao. O projeto tem estrutura preparada para ranking futuro, mas o jogo atual permanece local-first.

Pergunta:

Posso desafiar um amigo?

Resposta:

Sim. O jogo gera codigos de desafio para outra pessoa jogar a mesma sequencia de bandeiras.

Pergunta:

O jogo coleta dados pessoais?

Resposta:

A politica atual informa que o Flag Game nao solicita, armazena ou envia dados pessoais. Os recordes e progresso ficam no armazenamento local.
